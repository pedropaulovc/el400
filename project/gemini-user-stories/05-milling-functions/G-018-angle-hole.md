# US-G-018: Angle Hole (Linear)

**Manual Reference:** Section 9.1.4 Angle Hole Function
**Priority:** Low
**Category:** Milling Functions

## User Story
**As a** milling operator
**I want** to drill holes in a straight line at a specific angle
**So that** I can create angled linear patterns

## Acceptance Criteria
- [ ] **AC 18.1:** Press `Angle Hole` key (Display `AnGhoLE`).
- [ ] **AC 18.2:** Enter: Start Point (X, Y), `PitCh` (Distance between holes), `AnGLE`, `hoLES`.
- [ ] **AC 18.3:** Calculate and display Distance-to-Go.
