# US-G-019: Grid Function

**Manual Reference:** Section 9.1.5 Grid Function
**Priority:** Low
**Category:** Milling Functions

## User Story
**As a** milling operator
**I want** to drill a grid of holes
**So that** I can create matrix patterns (optionally tilted)

## Acceptance Criteria
- [ ] **AC 19.1:** Press `Grid` key.
- [ ] **AC 19.2:** Enter: Start (X, Y), `PitCh H` (X pitch), `PitCh Y` (Y pitch), `AnGLE` (Grid tilt), `hoLES H` (X count), `hoLES Y` (Y count).
- [ ] **AC 19.3:** Display Distance-to-Go for hole matrix.
