# US-G-025: Extended Setup Parameters

**Manual Reference:** Section 6.2 Parameters Setting
**Priority:** Low
**Category:** Configuration

## User Story
**As a** machine installer
**I want** to configure advanced machine parameters
**So that** the DRO behaves correctly for my specific machine geometry

## Acceptance Criteria
- [ ] **AC 25.1:** **Display Resolution (dP):** Set display increments (e.g., 1 vs 5 microns). Independent of Scale Resolution.
- [ ] **AC 25.2:** **Radius/Diameter (rAd/diA):** Set per axis. Diameter mode displays 2x movement.
- [ ] **AC 25.3:** **Sleep Timer (SLEEP t):** Set timeout in minutes (0-120). Display dims/sleeps.
- [ ] **AC 25.4:** **Keypad Lock (LoC):** Lock keyboard to prevent accidental changes.
- [ ] **AC 25.5:** **OEM Mode (oEn nod):** Password protected area.
