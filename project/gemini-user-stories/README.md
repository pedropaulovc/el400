# EL400 DRO Gemini User Stories

This directory contains user stories derived from the `EL400OpManual.md` (EL 400 Digital Readout System).

## Structure
- **01-foundation**: Startup, versioning, basic display.
- **02-core-operations**: ABS/INC, Zeroing, Units.
- **03-data-management**: SDM, Preset.
- **04-calculations**: Calculator, Center finding.
- **05-milling-functions**: PCD, Arc Contour, Grid, Angle Hole.
- **06-lathe-functions**: Tool Offset, Taper, Vectoring.
- **07-auxiliary**: Probe, Outputs, Serial.
- **08-configuration**: Setup parameters.
