# US-G-024: Pre-Set Depth (PSD)

**Manual Reference:** Section 9.3.4 Pre-Set Depth
**Priority:** Medium
**Category:** EDM Functions

## User Story
**As a** EDM operator
**I want** to trigger a relay when the electrode reaches a specific depth
**So that** the sparking process stops automatically

## Acceptance Criteria
- [ ] **AC 24.1:** Function available on Z axis (or X for 1-axis).
- [ ] **AC 24.2:** Press `PSD` key.
- [ ] **AC 24.3:** Enter Depth.
- [ ] **AC 24.4:** Relay output triggers when depth is reached.
- [ ] **AC 24.5:** Configurable sensitivity/delay (`SnStY 0` to `5`).
