# US-G-027: Touch Probe

**Manual Reference:** Section 10.1 Touch Probe Functions
**Priority:** Low
**Category:** Auxiliary

## User Story
**As a** operator
**I want** to use a touch probe to set datums
**So that** I can accurately locate workpiece edges and centers

## Acceptance Criteria
- [ ] **AC 27.1:** Config: `dro T` (Transmit) or `dro F` (Freeze).
- [ ] **AC 27.2:** Functions: `ProbE Ed` (Edge), `ProbE nd` (Midpoint), `inSidE`/`oUtSidE` (Center/Width).
- [ ] **AC 27.3:** On trigger, freeze display or set datum.
